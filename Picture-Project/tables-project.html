Tables pt. 1

In this other HTML project, I had to work with tables to make a time management tool featuring a calendar and a schedule differentiated between weekdays and weekends.

This furthered my learning on how to structure and style tables using HTML and CSS, an important step in improving my web design skills. I learned to make the data neat and presentable with the "table", "tr", "td", and "th" tags, while creating rows and columns.

This was quite an important skill when building layouts that deal with structured information, like calendars, plans, and price lists. Along with the structure in HTML, CSS was introduced to me for styling the tables. I implemented several visual enhancements, such as border, padding, background-color, and text-align.

Tables pt. 2

One of the main challenges of this project was how to highlight which day is a weekday or a weekend. Being able to use two different colored backgrounds for the weekend and weekday makes the readability of the schedule much easier for any end user.

One of the key challenges in this project was learning how to differentiate weekdays from weekends.
