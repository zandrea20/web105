About Me Project

My fifth project is a personal "Introduce Yourself" website, which shows information about myself, my hobbies, and my aspirations. This project was an exciting opportunity to bring together everything I had learned so far in HTML, CSS, and JavaScript, adding some advanced features too, such as multi-page navigation and image incorporation.

It has three pages: one about me, one about my hobbies, and another about my future aspirations. I added an easy navigation bar to navigate between these three pages with the use of HTML "nav" element.

That helped me learn how to structure a multi-page website and link those pages together by using anchor tags "a".

Then, I added images that would illustrate and help present my content on the page. I managed to align them using CSS and made it responsive by presenting pictures of my hobbies and personal achievements using HTML

This included using CSS in the creation of a clean and cohesive design by choosing the appropriate font and colors and styles of layouts, making the content easy to read and visually appealing.

This project gave me a basis on which to build a fully functioning website, solidifying even further the way in which to position content and imagery with text in order to help give navigation. This gave me further confidence in creating more complex real-world websites in the future.
